# React App

A very simple React app built with Vite, TypeScript, and React.

## Run Locally

```bash
npm install
npm run dev
```

## Run With Docker Hot Reload

```bash
docker compose up --build
```

Open:

```text
http://localhost:5173
```

## Run Production-Like Docker

```bash
docker compose -f docker-compose.prod.yml up --build
```

Open:

```text
http://localhost:5173
```
